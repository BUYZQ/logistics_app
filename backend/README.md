# Некст Backend

FastAPI бэкенд для системы управления логистикой пищевых товаров.

## Быстрый старт

### 1. Создать виртуальное окружение

```bash
cd backend
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # Linux/Mac
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Настроить переменные окружения

```bash
copy .env.example .env
# Отредактируйте .env: вставьте ключи SMS.ru и Яндекс SMTP
```

> **DEV MODE:** Если ключи не заполнены — коды OTP выводятся прямо в консоль сервера. Удобно для разработки без реальных SMS.

### 4. Запустить

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 5. Открыть Swagger UI

[http://localhost:8000/docs](http://localhost:8000/docs)

## API Endpoints

### Авторизация

| Метод | URL | Описание |
|-------|-----|----------|
| POST | `/auth/send-otp` | Отправить OTP на телефон или email |
| POST | `/auth/verify-otp` | Проверить OTP, получить JWT |
| GET | `/auth/me` | Профиль текущего пользователя |

### Сотрудники (только операторы)

| Метод | URL | Описание |
|-------|-----|----------|
| POST | `/employees/` | Добавить сотрудника |
| GET | `/employees/` | Список сотрудников склада |
| GET | `/employees/{id}` | Детали сотрудника |
| PATCH | `/employees/{id}/deactivate` | Деактивировать |
| GET | `/employees/warehouses/` | Список складов |

---

## Логика авторизации

```
1. Оператор добавляет сотрудника: POST /employees/
   → указывает телефон, email, имя, роль, склад
   → операторам автоматически присваивается номер (Оператор 1, 2, ...)

2. Сотрудник входит: POST /auth/send-otp
   → вводит телефон (+79141234567) или email
   → получает 6-значный код по SMS или email
   → код действителен 10 минут, максимум 5 попыток

3. Подтверждение: POST /auth/verify-otp
   → вводит код → получает JWT токен

4. Использование API: Authorization: Bearer <token>
```

## SMS-сервис — SMS.ru

1. Зарегистрируйтесь на [sms.ru](https://sms.ru)
2. Получите `API_ID` в личном кабинете
3. Вставьте в `.env`: `SMSRU_API_ID=ваш_ключ`

Стоимость: ~1.5–3 руб/SMS. При регистрации даётся 10 бесплатных SMS.

## Email-сервис — Яндекс SMTP

1. Создайте пароль приложения: [id.yandex.ru/security/app-passwords](https://id.yandex.ru/security/app-passwords)
2. Вставьте в `.env`:
   ```
   SMTP_USER=your@yandex.ru
   SMTP_PASSWORD=созданный_пароль
   ```
